# FSAA Tester (2003)
ToMMTi-Systems FSAA Tester

This is the original MSVC source code for the FSAA Tester available at https://www.tommti-systems.de that inspired Futuremark's 3DMark FSAA (Fullscreen Anti-Aliasing test).
